# Apresentações

Materiais comerciais e apresentações HTML exportáveis ficam aqui.

- `UzzApp_Apresentacao_Comercial.html`: apresentação comercial histórica movida do root.
- `docs/UzzApp_Apresentacao_Comercial_v2.html`: versão usada pelos scripts de exportação comercial atuais.

Use `npm run export:features-pdf` para o PDF de features ou os scripts `export:uzzapp-*` do `package.json` para apresentações comerciais específicas.
