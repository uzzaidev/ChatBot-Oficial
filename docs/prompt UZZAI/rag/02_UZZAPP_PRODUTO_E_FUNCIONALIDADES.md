# UzzApp - Produto e Funcionalidades

## O que e

UzzApp e uma plataforma SaaS de chatbot inteligente para WhatsApp Business API.
Cada cliente opera em ambiente isolado com credenciais proprias e configuracao propria de IA.

## Problemas que resolve

- Leads perdidos fora do horario comercial
- Sobrecarga com perguntas repetitivas
- Atendimento lento/inconsistente
- Dificuldade de escala sem aumentar equipe

## Funcionalidades principais

- Atendimento multimodal: texto, audio, imagem, PDF, documentos
- Memoria de conversa e contexto por contato
- RAG com base de conhecimento (pdf/txt)
- Fast track para FAQs
- Deteccao de repeticao de resposta
- Fluxos interativos sem codigo
- Escalonamento para humano com resumo
- Dashboard web + app mobile
- CRM integrado com kanban, tags e automacoes

## Seguranca

- Multi-tenant real por cliente
- Credenciais em vault criptografado
- RLS por cliente
- Validacao de webhook

## Limites

- Nao afirmar integracao custom como pronta sem validacao tecnica.
- Nao prometer SLA/feature especifica fora da base.
