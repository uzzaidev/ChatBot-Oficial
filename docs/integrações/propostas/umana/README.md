## 🎯 Como Usar Esta Documentação

### Passo 1: Entenda o Cliente

Antes de criar a proposta, identifique:
- **Setor:** Academia, imobiliária, clínica, etc.
- **Necessidades:** Vendas, suporte, qualificação?
- **Integrações:** Usa Tecnofit? Meta Ads? Outros sistemas?
- **Orçamento:** Qual faixa de investimento?

---

### Passo 2: Personalize a Proposta

1. **Abra** `PROPOSTA_COMERCIAL_FEATURES.md`
2. **Substitua** `[Nome do Cliente]` pelo nome real
3. **Escolha** features relevantes do checklist
4. **Adapte** os 6 benefícios principais conforme necessidade
5. **Ajuste** valores de investimento

---

### Passo 3: Use o Detalhamento

1. **Consulte** `DETALHAMENTO_FEATURES.md`
2. **Selecione** features relevantes para o cliente
3. **Adapte** a linguagem de benefícios
4. **Inclua** apenas o que faz sentido para o caso

---

### Passo 4: Crie o Design

Siga as dicas de design do documento:
- **Cores:** Azul tecnológico, roxo para IA, verde para checkmarks
- **Tipografia:** Sem serifa para títulos, legível para corpo
- **Espaçamento:** Muito espaço em branco
- **Ícones:** Use para facilitar leitura

---

### Passo 5: Revise o Checklist

Antes de finalizar, verifique:
- [ ] Capa tem nome do cliente?
- [ ] Frase de impacto na segunda página?
- [ ] Checklist visual de entregas?
- [ ] Destaque do produto principal?
- [ ] Investimento claro?
- [ ] Contatos e CTA no final?
- [ ] PDF não editável?
