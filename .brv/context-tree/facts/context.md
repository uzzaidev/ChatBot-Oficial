# Domain: facts

## Purpose
Contains repository-level factual knowledge, environment observations, and durable project metadata that do not belong to a narrower architecture or implementation domain.

## Scope
Included in this domain:
- Repository-wide factual observations
- Current tooling and knowledge-state checks
- Project-level metadata and constraints

Excluded from this domain:
- Detailed feature implementation notes
- Source-code architecture internals
- User personal preferences

## Ownership
Repository maintainers

## Usage
Use this domain for concise durable project facts and repository-level operational knowledge.
