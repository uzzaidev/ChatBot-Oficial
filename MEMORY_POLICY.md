# MEMORY POLICY — ChatBot-Oficial

## Permitido memorizar
- Decisões arquiteturais e trade-offs
- Padrões de código e convenções do projeto
- Bugs recorrentes e suas soluções
- Comandos úteis e fluxos de trabalho
- Fluxos críticos do pipeline (14 nós)
- Regras de multi-tenancy e isolamento de dados
- Integrações externas e seus limites
- Tech debt conhecida e riscos identificados

## Proibido memorizar
- API keys, tokens, senhas ou qualquer credencial
- Dados de clientes ou usuários finais
- Conteúdo de arquivos .env ou .env.local
- Contratos e informações financeiras
- Dados pessoais identificáveis (CPF, telefone real, email de cliente)
- Segredos do Supabase Vault
